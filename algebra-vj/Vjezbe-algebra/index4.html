Uvod u HTML

HTML je jezik za obiljezavanje koji koristimo za izradu web stranica.

Njime definiramo strukturu i sadrzaj web stranica.

Povijest HTML-a

HTML je nastao pocetkom 1990-ih godina.

Uredivanje teksta

Tekst je naajcesci sadrzaj web stranica.

Da bi dobio odgovarajucu ulogu a stranici bio prikazan na odredeni nacin uredujemo ga HTML-om.

HTML atributi

Unutar pocetne oznake elemenata, nakon naziva elementa, po potrebi pisemo i atribute.

Atributi internetskim preglednicima daju dodatne informacije o elementu. O samom atributu ovisi o kojim informacijama je rijec.

Atribut se sastoji od imena i vrijednosti atributa.
